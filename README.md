# MPV-Miners
### 太坊矿池代理WEB管理系统,SSL/TCP转发混淆加密,支持抽水模式

### 支持矿池
- E池
- 鱼池
- H池
- 风池
- 蚂蚁
- 币印
- 欧意
- 2miners

### 功能

- 一键添加矿池节点
- 中转+抽水模式
- 自定义抽水设置
- 实时算力日志
- 在线更新节点
- 开箱即用
- 远程设置


### 使用
- 运行软件安装安装运行
- 打开浏览器访问服务器18848端口 http://ip:18848
- 默认登录密钥：88888888
